Ejercicio 2

Dada una lista de enteros se ordenan por diferentes algoritmos y se devuelve.
(Principalmente también se iban a mostrar los tiempos de ejcución de cada algoritmo pero para que empiece a haber diferencias notorias en tiempo tendríamos que dar de entrado una lista de tamaño entero, y como introducimos la lista por URL lo obviaremos.)

Ejemplo de uso: ordena/11,23,8,2,41 Probar

Ejercicio 3

La Criba de Eratóstenes es un sencillo algoritmo que permite encontrar todos los números primos menores de un número natural dado.

Ejemplo de uso: /cribaEratostenes/27 Probar

Ejercicio 4

Dado un número n calcula el n-ésimo número de la sucesión de Fibonacci

Ejemplo de uso: /fibonacci/7 Probar

Ejercicio 5

Dada una cadena de parentesis '[' y ']' calcula si dicha secuencia está balanceada es decir, que se componga de parejas de corchetes de apertura y cierre correctamente anidados.

Ejemplo de uso: /checkParentesis/[][[]]][] Probar

Ejercicio 6

Dada una cadena de entrada determina si existe un email, una tarjeta de crédito o una incial de apellido y lo devuelve.

Ejemplo de uso: /regex/paco@gmail.com Probar
